## How to play
| Computer | Space Invaders                                 |
|----------|------------------------------------------------|
| NUMPAD1  | Insert credits                                 |
| NUMPAD7  | Start game in 1 player mode                    |
| NUMPAD9  | Start a game in 2 player mode                  |
| NUMPAD4  | Move left                                      |
| NUMPAD6  | Move right                                     |
| NUMPAD8  | shoot                                          |
